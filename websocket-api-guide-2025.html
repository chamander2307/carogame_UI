Kết nối WebSocket

WebSocket URL:

ws://localhost:8080/ws?token=Bearer {JWT_TOKEN}

Protocol: STOMP (Simple Text Oriented Messaging Protocol)

Authentication: JWT Token trong query parameter

Lưu ý quan trọng: Tất cả WebSocket connections phải bao gồm JWT token hợp lệ để xác thực. Token sẽ được validate tự động khi kết nối.

Xác thực JWT

JavaScript Connection Example

// Kết nối WebSocket với JWT token const token = localStorage.getItem('accessToken'); const socket = new WebSocket(`ws://localhost:8080/ws?token=Bearer ${token}`); const stompClient = Stomp.over(socket); // Headers for STOMP connection const headers = { 'Authorization': `Bearer ${token}` }; // Connect với error handling stompClient.connect(headers, function(frame) { console.log('Connected:', frame); // Subscribe to topics sau khi connected subscribeToRoomEvents(roomId); }, function(error) { console.error('Connection error:', error); } );

Client → Server Endpoints

Các destinations mà client có thể gửi messages đến server:

SEND Tham gia phòng

/app/room/{roomId}/join
Tham gia vào một phòng game cụ thể. Player sẽ được thêm vào phòng với trạng thái NOT_READY.

Payload:

{}

Response Events:

// Broadcast đến tất cả players trong phòng Topic: /topic/room/{roomId} { "type": "PLAYER_JOINED", "data": { "playerId": 123, "playerName": "john_doe", "readyState": "NOT_READY", "joinedAt": "2025-01-08T10:30:00Z" } }

SEND Đánh dấu sẵn sàng (Enhanced)

/app/room/{roomId}/ready
Enhanced Feature: 2-step ready system. Player click "Mark Ready" để chuyển từ NOT_READY → READY. Game tự động start khi cả 2 players đều READY.

Payload:

{}

Response Events:

// Player ready event { "type": "PLAYER_READY", "data": { "playerId": 123, "playerName": "john_doe", "readyState": "READY" } } // Auto game start khi cả 2 players ready { "type": "GAME_STARTED", "data": { "gameState": "IN_PROGRESS", "gameStartedAt": "2025-01-08T10:30:00Z", "currentTurn": "X", "players": [ { "id": 123, "symbol": "X", "readyState": "IN_GAME" } ] } }

SEND Thực hiện nước đi

/app/game/{roomId}/move
Thực hiện một nước đi trong game Caro. Server sẽ validate tính hợp lệ của nước đi và kiểm tra điều kiện thắng.

Payload:

{ "xPosition": 7, "yPosition": 8 }

Response Events:

// Move made event Topic: /topic/game/{roomId}/move { "roomId": 123, "matchId": 456, "xPosition": 7, "yPosition": 8, "playerId": 123, "playerSymbol": "X", "moveNumber": 15, "nextTurnPlayerId": 456, "gameState": "IN_PROGRESS", "gameResult": "ONGOING", "winnerId": null, "board": [[0,1,2,...], ...], "isValidMove": true, "message": "Move successful", "timestamp": "2025-01-08T10:35:00.123456" } // Game win event (nếu thắng) { "type": "GAME_WON", "data": { "winnerId": 123, "winnerName": "john_doe", "winningSequence": [[7,8], [8,9], [9,10], [10,11], [11,12]], "gameEndReason": "WIN", "gameEndedAt": "2025-01-08T10:45:00Z" } }

SEND Đầu hàng

/app/room/{roomId}/surrender
Player đầu hàng và tự động thua game. Game sẽ kết thúc ngay lập tức.

Payload:

{}

Response Events:

// Game ended by surrender Topic: /topic/room/{roomId} { "type": "GAME_ENDED", "data": { "gameResult": "SURRENDER", "winnerId": 456, "loserId": 123, "endedAt": "2025-01-08T10:45:00Z", "endReason": "Player surrendered" } }

SEND Yêu cầu rematch (Enhanced 2-step)

/app/room/{roomId}/rematch/request
Enhanced Feature: Step 1 của 2-step rematch process. Player request rematch sau khi game kết thúc.

Payload:

{}

Response Events:

{ "type": "REMATCH_REQUESTED", "data": { "requesterId": 123, "requesterName": "john_doe", "message": "john_doe wants a rematch!", "requestedAt": "2025-01-08T10:50:00Z" } }

SEND Chấp nhận rematch (Enhanced 2-step)

/app/room/{roomId}/rematch/accept
Enhanced Feature: Step 2 của 2-step rematch process. Player accept rematch request → tự động tạo room mới.

Payload:

{}

Response Events:

// Accept event { "type": "REMATCH_ACCEPTED", "data": { "accepterId": 456, "accepterName": "jane_doe", "message": "jane_doe accepted the rematch!" } } // Auto tạo room mới { "type": "REMATCH_CREATED", "data": { "newRoomId": 789, "message": "New rematch room created! Redirecting...", "redirectUrl": "/gameroom/789" } }

SEND Gửi tin nhắn chat

/app/room/{roomId}/chat
Gửi tin nhắn chat trong phòng game. Tin nhắn sẽ được broadcast đến tất cả players trong phòng.

Payload:

{ "content": "Hello everyone! Good luck! 🎮" }

Response Events:

Topic: /topic/room/{roomId}/chat { "type": "CHAT_MESSAGE", "data": { "senderId": 123, "senderName": "john_doe", "content": "Hello everyone! Good luck! 🎮", "timestamp": "2025-01-08T10:32:00Z" } }

SEND Rời phòng

/app/room/{roomId}/leave

Payload:

{}

Response Events:

{ "type": "PLAYER_LEFT", "data": { "playerId": 123, "playerName": "john_doe", "leftAt": "2025-01-08T10:40:00Z", "gameEndReason": "LEAVE" } }

Server → Client Subscriptions

Các topics mà client cần subscribe để nhận events từ server:

SUBSCRIBE Room Events

/topic/room/{roomId}
Subscribe để nhận tất cả events liên quan đến phòng game (join, leave, ready, game state changes, rematch events).

Event Types:

// Các loại events có thể nhận được: - ROOM_UPDATE // Cập nhật thông tin phòng - PLAYER_JOINED // Player tham gia phòng - PLAYER_LEFT // Player rời phòng - PLAYER_READY // Player đánh dấu ready (Enhanced) - GAME_STARTED // Game bắt đầu (Auto when both ready) - GAME_ENDED // Game kết thúc - REMATCH_REQUESTED // Yêu cầu rematch (Enhanced) - REMATCH_ACCEPTED // Chấp nhận rematch (Enhanced) - REMATCH_CREATED // Room mới được tạo (Enhanced)

JavaScript Subscribe Example

// Subscribe to room events stompClient.subscribe(`/topic/room/${roomId}`, function(message) { const event = JSON.parse(message.body); switch(event.type) { case 'PLAYER_JOINED': handlePlayerJoined(event.data); break; case 'PLAYER_READY': handlePlayerReady(event.data); break; case 'GAME_STARTED': handleGameStarted(event.data); break; case 'REMATCH_REQUESTED': showRematchAcceptButton(event.data); break; case 'REMATCH_CREATED': redirectToNewRoom(event.data.newRoomId); break; default: console.log('Unknown event:', event); } });

SUBSCRIBE Game Moves

/topic/game/{roomId}/move
Subscribe để nhận thông tin về các nước đi trong game (moves, win, draw).

Event Types:

// Game move events: - MOVE_MADE // Nước đi được thực hiện - GAME_WON // Game có người thắng - GAME_DRAW // Game hòa - INVALID_MOVE // Nước đi không hợp lệ

SUBSCRIBE Game End Events

/topic/game/{roomId}/end
Subscribe để nhận thông báo khi game kết thúc (win, draw, surrender). Topic này chỉ được broadcast khi game state = FINISHED.

Message Format:

{ "roomId": 123, "matchId": 456, "gameState": "FINISHED", "gameResult": "WIN", // WIN, DRAW, SURRENDER "winnerId": 123, "board": [[0,1,2,...], ...], "message": "Game won", "timestamp": "2025-01-08T10:35:00.123456" }

SUBSCRIBE Global Room Updates

/topic/room-updates
Subscribe để nhận thông báo về tất cả room updates toàn cục (room created, deleted, player count changes).

Message Format:

{ "type": "ROOM_UPDATED", "data": { "roomId": 123, "roomName": "My Room", "playerCount": 2, "maxPlayers": 2, "status": "WAITING", "updateType": "PLAYER_JOINED" } }

SUBSCRIBE Chat Messages

/topic/room/{roomId}/chat
Subscribe để nhận tin nhắn chat trong phòng game.

Message Format:

{ "type": "CHAT_MESSAGE", "data": { "senderId": 123, "senderName": "john_doe", "content": "Message content", "timestamp": "2025-01-08T10:32:00Z" } }

SUBSCRIBE Personal Notifications

/user/queue/notifications
Subscribe để nhận thông báo cá nhân (friend requests, game invitations, system messages).

Notification Types:

// Personal notification events: - FRIEND_REQUEST // Lời mời kết bạn - GAME_INVITATION // Lời mời chơi game - SYSTEM_MESSAGE // Thông báo hệ thống - USER_STATUS_CHANGE // Thay đổi trạng thái bạn bè

// 1. Players join room /app/room/{roomId}/join → PLAYER_JOINED event (readyState: NOT_READY) // 2. Players mark ready /app/room/{roomId}/ready → PLAYER_READY event (readyState: READY) // 3. Auto game start when both ready → GAME_STARTED event (gameState: IN_PROGRESS) // 4. Players make moves /app/room/{roomId}/move → MOVE_MADE events // 5. Game ends → GAME_WON/GAME_DRAW event // 6. Error Handling

ERROR Error Response Format

Error Message Format:

{ "type": "ERROR", "data": { "errorCode": "ROOM_FULL", "message": "Room is full. Cannot join.", "details": { "roomId": 123, "currentPlayers": 2, "maxPlayers": 2 }, "timestamp": "2025-01-08T10:35:00Z" } }

Common Error Codes:

ROOM_NOT_FOUND // Phòng không tồn tại ROOM_FULL // Phòng đã đầy ALREADY_IN_ROOM // Đã ở trong phòng khác GAME_IN_PROGRESS // Game đang diễn ra INVALID_MOVE // Nước đi không hợp lệ NOT_YOUR_TURN // Không phải lượt của bạn AUTHENTICATION_ERROR // Lỗi xác thực JWT PERMISSION_DENIED // Không có quyền truy cập PLAYER_NOT_READY // Player chưa ready REMATCH_NOT_AVAILABLE // Không thể rematch
Lưu ý: Client nên handle tất cả error cases và hiển thị thông báo phù hợp cho user. Error events được gửi trực tiếp đến client gây ra lỗi, không broadcast.

Best Practices

Connection Management

  • Always validate JWT token trước khi connect
  • Implement reconnection logic với exponential backoff
  • Handle connection lost gracefully
  • Clean up subscriptions khi disconnect

Event Handling

  • Subscribe đến tất cả required topics sau khi connect
  • Handle tất cả event types (không bỏ qua)
  • Implement proper error handling cho mỗi event
  • Update UI immediately khi nhận events

Game State Management

  • Track game state locally để validate actions
  • Implement ready system properly (2-step)
  • Handle rematch flow correctly (2-step)
  • Validate moves client-side trước khi send

Security

  • Never expose JWT token trong logs
  • Refresh token khi gần expire
  • Validate server responses trước khi process
  • Implement rate limiting cho actions

Performance

  • Minimize số lượng subscriptions
  • Debounce rapid user actions
  • Use efficient DOM updates
  • Cache data locally khi có thể

Debugging

  • Log tất cả WebSocket events với timestamps
  • Include request/response correlation IDs
  • Monitor connection health regularly
  • Implement comprehensive error logging

Complete Client Implementation Example

class CaroWebSocketClient { constructor(token, roomId) { this.token = token; this.roomId = roomId; this.stompClient = null; this.subscriptions = []; this.isConnected = false; } connect() { const socket = new WebSocket(`ws://localhost:8080/ws?token=Bearer ${this.token}`); this.stompClient = Stomp.over(socket); this.stompClient.connect({}, frame => this.onConnected(frame), error => this.onError(error) ); } onConnected(frame) { console.log('Connected:', frame); this.isConnected = true; // Subscribe to all required topics this.subscribeToTopics(); // Join room automatically this.joinRoom(); } subscribeToTopics() { // Room events this.subscriptions.push( this.stompClient.subscribe(`/topic/room/${this.roomId}`, message => this.handleRoomEvent(JSON.parse(message.body)) ) ); // Game moves this.subscriptions.push( this.stompClient.subscribe(`/topic/game/${this.roomId}/move`, message => this.handleMoveEvent(JSON.parse(message.body)) ) ); // Game end events this.subscriptions.push( this.stompClient.subscribe(`/topic/game/${this.roomId}/end`, message => this.handleGameEndEvent(JSON.parse(message.body)) ) ); // Chat messages this.subscriptions.push( this.stompClient.subscribe(`/topic/room/${this.roomId}/chat`, message => this.handleChatEvent(JSON.parse(message.body)) ) ); // Personal notifications this.subscriptions.push( this.stompClient.subscribe('/user/queue/notifications', message => this.handleNotification(JSON.parse(message.body)) ) ); } handleRoomEvent(event) { switch(event.type) { case 'PLAYER_JOINED': this.updatePlayerList(event.data); break; case 'PLAYER_READY': this.updatePlayerReadyState(event.data); break; case 'GAME_STARTED': this.startGame(event.data); break; case 'REMATCH_REQUESTED': this.showRematchAcceptButton(event.data); break; case 'REMATCH_CREATED': this.redirectToNewRoom(event.data.newRoomId); break; } } // Enhanced methods for new features markReady() { if (this.isConnected) { this.stompClient.send(`/app/room/${this.roomId}/ready`, {}, '{}'); } } requestRematch() { if (this.isConnected) { this.stompClient.send(`/app/room/${this.roomId}/rematch/request`, {}, '{}'); } } acceptRematch() { if (this.isConnected) { this.stompClient.send(`/app/room/${this.roomId}/rematch/accept`, {}, '{}'); } } surrender() { if (this.isConnected) { this.stompClient.send(`/app/room/${this.roomId}/surrender`, {}, '{}'); } } // Handle game end events (win, draw, surrender) handleGameEndEvent(event) { console.log('Game ended:', event); switch(event.gameResult) { case 'WIN': this.showGameResult('Game Over', `Player ${event.winnerId} wins!`); break; case 'DRAW': this.showGameResult('Game Over', 'It\'s a draw!'); break; case 'SURRENDER': this.showGameResult('Game Over', `Player ${event.winnerId} wins by surrender!`); break; } this.disableGameBoard(); this.showRematchButton(); } disconnect() { if (this.stompClient && this.isConnected) { // Clean up subscriptions this.subscriptions.forEach(sub => sub.unsubscribe()); this.stompClient.disconnect(); this.isConnected = false; } } } // Usage const client = new CaroWebSocketClient(jwtToken, roomId); client.connect();
